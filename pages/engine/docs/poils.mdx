# Introduction

## What is the Polis?
Polis is a modular application development engine composed of a series of toolchains and libraries, designed to simplify modular application development. Users can leverage this engine to develop various applications, which can be entirely or partially built on a modular blockchain network.

## Polis Architecture?

* [Schema](/engine/docs/poils/schemas) provides abstractions specifically designed for storing and retrieving data from a on-chain permanent storage.
* [CLI](/engine/docs/poils/cli) is used for building and managing a Poils Project.
* [Client](/engine/docs/poils/client) is implement the synchronization protocol and a client-side cache to replicate Store tables, along with the necessary infrastructure to send transactions to the World.
* [Create-Obelisk](/engine/docs/poils/create-obelisk) is a CLI tool that helps you create Poils projects.
